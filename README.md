## 功能介绍

```properties
管理员登录后可对系统进行全面管理，管理员主要实现的功能模块包括个人中心、用户管理、笔记分享管理、笔记类型管理以及系统管理五大部分，


用户进入本系统可查看系统信息，包括首页、笔记分享，注册登录后主要功能模块包括个人中心、笔记分享管理、笔记类型管理以及我的收藏管理，
```
